# Тестовое задание
## Запуск для проверки работоспособности
#### 1. Все приложение собирается в Docker. Необходимо запустить docker-compose.yml. 
#### 2. Запуск приложения из среды разработки. Запуск через класс TestApplication. Необходимо заранее создать пустую базу в PostgresSQL. Изменить настройки подключения в файле конфигурации - "resources/application.properties"
Запуск фронта производится из файла "/resources/front/index.html". Для проверки работоспособности в БД создаются 2 тестовые учетные записи - логин: admin, пароль :admin; логин: user, пароль - password.
