# Тестовое задание
## Запуск для проверки работоспособности
### 1. Docker
    Необходимо запустить файл "docker-compose.yml", который лежит в корне проекта.
### 2. Запуск приложения из среды разработки
    Запуск через класс TestApplication. Необходимо заранее создать пустую базу в PostgresSQL. 
    Изменить настройки подключения в файле конфигурации - "resources/application.properties".
#### Общая информация для обоих вариантов запуска
    Запуск фронта производится из файла "/resources/front/index.html". 
    Для проверки работоспособности в БД создаются 2 тестовые учетные записи:
     1. логин: admin, пароль :admin
     2. логин: user, пароль - password
